
# 🌐 Swastik Care Website – README

## 📌 Project Overview

**Swastik Care** is a responsive healthcare website designed to provide users with easy access to medical services, information, and support. The project focuses on clean UI design, accessibility, and user-friendly navigation.

This website showcases a modern front-end layout suitable for clinics, hospitals, or healthcare service providers.

---

## 🚀 Features

* Responsive design (works on mobile, tablet, desktop)
* Clean and modern user interface
* Easy navigation between sections
* Informational pages for healthcare services
* Contact section for user inquiries
* Structured layout for scalability

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** – Basic interactivity (if used)

## 💡 How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Double-click on `index.html`
4. The website will open in your browser
